# MEEP 综合项目

本目录包含完整的 MEEP 仿真项目，每个项目都包含：
- 设计目标与参数
- 完整的仿真代码
- 结果分析与可视化
- 优化建议

## 项目列表

| 编号 | 项目 | 描述 |
|------|------|------|
| 01 | 波分复用器 | 基于环形谐振腔的 DWDM 滤波/分路器 |
| 02 | 光隔离器 | 磁光材料 + 光子晶体非互易传输 |
| 03 | 超表面透镜 | 梯度超表面的波前调控 |

## 如何提交你的项目

1. 在 `projects/` 下创建以项目名命名的文件夹
2. 包含一个主要的 `.py` 仿真脚本
3. 可选：README 说明设计思路
4. 提交 PR！
